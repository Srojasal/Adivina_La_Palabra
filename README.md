#Guess the Word

Console-based game in Java (NetBeans) where the player tries to guess words letter by letter, with attempt tracking and visual hints.

Implements row/column logic and a modular structure in paqueteproyecto2.
Includes build with build.xml, compiled classes in classes, and a NetBeans project in nbproject.
Ideal as an educational project for basic programming: loops, conditionals, methods, and input/output handling.
